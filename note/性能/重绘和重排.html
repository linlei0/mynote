# 页面生成过程

  1. HTML被HTML解析器解析成DOM树
  2. CSS被CSS解析器解析成CSSOM树
  3. DOM树和CSSOM树生成Render Tree也就是渲染树
  4. 生成布局，浏览器在屏幕上画出渲染树中的所有节点
  5. 将布局绘制在屏幕上，显示整个页面

其中第四步和第五步是最耗时的，这两步合起来就是我们常说的渲染。
